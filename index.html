Welcome to Log4jToXml project page

The main purpose of this page is to inform you about the project for subject PB138 - Modern markup languages and their application, Faculty of Informatics, Masaryk University, Brno, spring 2015.

The aim of the project

The main aim of this project is to create a fully functional executable java application which will be used as a converter for the log4j configuration. The application will be able to convert the log4j configuration between two formats commonly used for storing log4j - properties and XML, both ways. The final application will be executable from the command line and from the Java code (probably jar executable). The final project should have an user friendly GUI before it will be published.

Basic project schema

The project consists of four main parts, each one is managed by one person. The back end is handled by Frederik Hudák ( log4j.properties --> XML ) and Martin Štefanko (XML --> log4j.properties). The front end (GUI and executable application) is managed mainly by Lukáš Linhart, with help of David Kratochvíl, who is mainly responsible for the application testing.
